# Linux Security Audit Framework (Single-Node)

A defensive cybersecurity audit framework for Linux systems using Kali Linux tools.
This project performs network analysis, system hardening assessment, credential auditing,
malware detection, and continuous security monitoring on a single machine.

## Tools Used
- Nmap
- Lynis
- John the Ripper
- Chkrootkit
- Auditd

## Ethical Note
This project performs only authorized, defensive security assessments on the author's own system.
